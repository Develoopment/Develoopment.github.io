RTOS for ARM Cortex-M

A lightweight Real-Time Operating System developed from scratch for ARM Cortex-M microcontrollers.

// project_overview()

This project involved developing a lightweight Real-Time Operating System (RTOS) from the ground up specifically for ARM Cortex-M microcontrollers. The RTOS supports task scheduling, inter-task communication, and resource management.

Key features include preemptive multitasking, semaphores, message queues, and memory management, all designed with minimal overhead for resource-constrained embedded systems.

Core Components

  • Scheduler: Priority-based preemptive scheduler with round-robin for equal priority tasks
  • Memory Management: Static memory allocation with fixed-size blocks
  • Synchronization: Semaphores, mutexes, and message queues
  • Context Switching: Hand-optimized ARM assembly for fast task switching

Performance Metrics

  • Context switch time: < 5µs @ 72MHz
  • Memory footprint: < 8KB (including kernel)
  • Maximum tasks: 32 (configurable)
  • Priority levels: 8 (configurable)

Development Process

The RTOS was developed using a test-driven approach, with each component tested in isolation on an STM32F4 Discovery board before integration.

Key challenges included optimizing context switching time and minimizing interrupt latency while maintaining a clean, maintainable codebase.

Tools Used

  • GCC ARM Embedded Toolchain
  • OpenOCD for debugging
  • STM32CubeIDE for project setup
  • Git for version control
